# 🚀 Features

## 🔐 Authentication

- User signup/login
- JWT-based authentication
- Secure password hashing

## 💰 Transactions

- Add income/expense
- Edit/delete transactions
- Category selection
- Notes/description

## 🏦 Accounts

- Multiple accounts (cash, bank)
- Balance tracking

## 📊 Dashboard

- Total income/expense
- Category-wise breakdown
- Monthly trends

## 🎯 Budgeting

- Set monthly budgets
- Track usage
- Budget alerts

## 🎯 Financial Goals

### Goal Creation

- Create goals (Vacation, Gadget, Emergency Fund, etc.)
- Set:
  - Target amount
  - Target date
  - Monthly contribution (optional)

### Goal Tracking

- Track saved amount vs target
- Show progress percentage
- Visual progress bar

### Feasibility Analysis

- Compare savings vs time remaining
- Show:
  - "On track"
  - "At risk"
  - "Not achievable"

### Insights

- Required monthly savings to meet goal
- Suggest adjustments

## 🔍 Filters & Search

- Filter by date
- Filter by category
- Search by description

## 🔔 Notifications (Optional)

- Budget exceeded alerts
- Goal progress alerts
- Deadline reminders

## 🔮 Future (AI Phase)

- Auto categorization
- Spending prediction
- Goal feasibility prediction (ML)
- Smart recommendations
