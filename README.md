# 💡 AI-Powered Personal Finance Tracker

## 🚀 Overview

An end-to-end **personal finance management platform** built using the MERN stack, designed to help users:

- Track income and expenses
- Analyze spending patterns
- Set and achieve financial goals
- (Future) Leverage AI for predictions and smart recommendations

---

## 🎯 Problem Statement

Most individuals:

- Do not consistently track expenses
- Lack visibility into spending habits
- Struggle to plan and achieve financial goals

---

## 💡 Solution

This application provides:

- A centralized platform to manage finances
- Visual insights into spending behavior
- Goal-based financial planning (e.g., vacation, gadgets)
- A foundation for future AI-driven financial intelligence

---

## 🧱 Tech Stack

### Frontend

- React (TypeScript)
- Tailwind CSS
- Axios

### Backend

- Node.js
- Express.js
- MongoDB (Mongoose)

### Future Integration

- Python (FastAPI for ML services)
- Scikit-learn / PyTorch

---

## ✨ Features

### 🔐 Authentication

- Secure user signup/login (JWT-based)

### 💰 Transactions

- Add, edit, delete income/expenses
- Categorization and descriptions

### 🏦 Accounts

- Multiple account support
- Balance tracking

### 📊 Dashboard

- Expense breakdown (category-wise)
- Monthly trends
- Income vs expense analysis

### 🎯 Budgeting

- Monthly budget limits
- Usage tracking and alerts

### 🎯 Financial Goals

- Create goals (Vacation, Laptop, Emergency Fund)
- Track savings progress
- Feasibility analysis (on-track / at-risk)
- Monthly savings recommendations

### 🔍 Filters & Search

- Filter by date/category
- Search transactions

---

## 🤖 Future Scope (AI Integration)

- Auto expense categorization (NLP)
- Spending prediction (time-series)
- Anomaly detection (fraud/waste alerts)
- Goal feasibility prediction
- Personalized financial recommendations
